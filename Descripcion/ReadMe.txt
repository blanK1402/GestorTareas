El archivo Tasks.txt contiene tareas para posteriormente cargarlas al programa
Cada linea representa una tarea
Tiene formato
ID, Descripcion, completada(Boolean), prioridad(baja, media, alta)
Ejemplo:
1,Comprar pan,false,alta


Estructura de los paquetes

-Model:
Aqui van las clases del proyecto
-View
Interfaz del proyecto
-Util
Funciones utiles
-Contoller
Intermediario entre la logica y la interfaz
-Main
Inicia la aplicacion
